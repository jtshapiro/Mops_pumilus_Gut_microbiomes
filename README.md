Data and scripts to perform analysis for Shapiro et al. (submitted) "Gut microbiomes of the little free-tailed bat (Mops pumilus) 
show high prevalence of potential bacterial pathogens but limited responses to land cover." 

Scripts are numbered in the order they should be run. Objects created in one script may be required to carry out analysis in the following scripts.

Each script is sub-divided into sections and fully commented.

All scripts were tested in for R version 4.1.2.
